# AI Context Map

This folder stores durable project context for AI coding agents. Keep it short,
current, and operational.

## Read Order

1. `AGENTS.md`
2. `README.md`
3. `docs/ai/project-context.md`
4. `docs/ai/rules.md`
5. `docs/ai/task-brief-template.md` when starting a scoped feature

## Maintenance Rule

Update these docs in the same change when a task changes project structure,
commands, environment variables, API contracts, design conventions, or product
assumptions.
